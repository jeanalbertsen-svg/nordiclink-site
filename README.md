# Nordic Link Consulting (React + Vite)

## Run locally

1. Install dependencies

```bash
npm install
```

2. Start dev server

```bash
npm run dev
```

Open the URL printed in the terminal.

## Build + preview

```bash
npm run build
npm run preview
```

## Customize
- Update email in `src/pages/Contact.jsx`
- Replace brand mark/text in `src/components/Header.jsx`
- Adjust colors in `src/styles/global.css`
